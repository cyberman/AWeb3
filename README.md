# AWeb3

AWeb3 is a modern web browser implementation for AmigaOS, focusing on standards compliance and performance.

## Overview

This repository contains the source code and development files for AWeb3, a next-generation web browser for the Amiga platform.

## Development Status

Currently in early development phase.
