HTML 3.2 Compatibility Test

This page tests AWeb's compatibility with the HTML 3.2 standard (W3C Recommendation, January 1997). Each section demonstrates supported features and clearly marks any incompatibilities or limitations.

Document Structure

Fully Supported

All HTML 3.2 document structure elements are fully supported:

  • <HTML> - Document root element
  • <HEAD> - Document head section
  • <BODY> - Document body with attributes:
    • BGCOLOR - Background color
    • TEXT - Text color
    • LINK - Link color
    • VLINK - Visited link color
    • ALINK - Active link color
    • BACKGROUND - Background image
  • <TITLE> - Document title
  • <BASE> - Base URL for relative links
  • <ISINDEX> - Simple search form
  • <META> - Meta information
  • <LINK> - Related resources

    Compatibility Notes

    HTML 3.2 Features Fully Supported

    • All document structure elements (HTML, HEAD, BODY, TITLE, BASE, META, LINK, ISINDEX)
    • All text formatting elements (headings, paragraphs, line breaks, horizontal rules)
    • All preformatted text elements (PRE, LISTING, XMP)
    • Text alignment (CENTER, DIV with ALIGN, P with ALIGN)
    • All text style elements (physical, logical, monospace, size, positioning)
    • Inserted and deleted text (INS, DEL - HTML 4.0)
    • Font styling (FONT, BASEFONT with SIZE, COLOR, FACE)
    • All list types (UL, OL, DL, DIR, MENU) with all attributes
    • Hyperlinks and anchors (A with HREF, NAME, TARGET)
    • Images (IMG with all standard attributes)
    • Client-side image maps (MAP, AREA with all SHAPE types)
    • Tables (TABLE, CAPTION, TR, TD, TH with all HTML 3.2 attributes)
    • Forms (FORM, INPUT, SELECT, OPTION, TEXTAREA with all HTML 3.2 attributes)
    • Character entities (named and numeric)

    HTML 3.2 Features Requiring Tolerant Mode

    • NOBR - No-break text (Netscape extension)
    • WBR - Word break opportunity (Netscape extension)
    • BLINK - Blinking text (Netscape extension)

    HTML 3.2 Limitations

    • Some advanced table features may have rendering quirks
    • Complex nested forms may not be fully supported
    • Some image map coordinate calculations may have minor issues

    HTML 3.2 vs HTML 4.0

    This page tests HTML 3.2 specifically. For HTML 4.0 features, see html4_test.html.
