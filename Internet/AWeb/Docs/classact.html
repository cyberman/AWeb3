ReAction

AWeb uses the ReAction (formerly ClassAct) GUI toolkit.

ReAction is a set of BOOPSI gadget and image classes, designed to speed up and simplify the development of GUI applications. ReAction is integrated into all modern version of the Amiga operating system software. Alternatively, users of older versions can use the original ClassAct available from Aminet https://www.aminet.net/dev/gui/classact33.lha
