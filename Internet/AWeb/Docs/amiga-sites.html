Browser Compatibility Notes

Most of these sites are designed to work with basic HTML browsers and should be compatible with AWeb's HTML 3/4 support. However, some modern features like JavaScript, CSS3, or complex layouts may not display correctly.

If you encounter display issues, try:

  • Disabling images in AWeb settings
  • Using the text-only mode if available
  • Checking the site's mobile or basic HTML version

Adding to Your Hotlist

You can add any of these links to your AWeb hotlist by:
  1. Right-clicking on a link and selecting "Add to hotlist"
  2. Using the Hotlist / Add to hotlist menu item
  3. Pressing the add to hotlist button while viewing a page
