Amiga Websites

Popular Amiga Sites

This page contains links to popular Amiga-related websites that are generally compatible with Amiga browsers like AWeb. These sites typically use simple HTML layouts that work well with limited HTML 3/4 support.

Note Despite what it says above, many of the sites listed here are no longer compatible with AWeb as they have been updated to more modern standards in the intervening years.

  • Aminet - The largest Amiga software archive
  • Amiga Forever - The Official home of Amiga emulation
  • AmigaOS 3 Developer - Home of the Native Developer Kit and documentation

  • AmigaWorld.net Forums - Main community forums
  • Abime.net - Next generation Amiga news

Browser Compatibility Notes

Most of these sites are designed to work with basic HTML browsers and should be compatible with AWeb's HTML 3/4 support. However, some modern features like JavaScript, CSS3, or complex layouts may not display correctly.

If you encounter display issues, try:

  • Disabling images in AWeb settings
  • Using the text-only mode if available
  • Checking the site's mobile or basic HTML version

Adding to Your Hotlist

You can add any of these links to your AWeb hotlist by:
  1. Right-clicking on a link and selecting "Add to hotlist"
  2. Using the Hotlist / Add to hotlist menu item
  3. Pressing the add to hotlist button while viewing a page
