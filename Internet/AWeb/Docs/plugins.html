Image Format Plugins

AWeb plugins extend the functionality of AWeb by providing support for additional file formats and protocols. The following image format plugins are available for AWeb:

Image Plugins 

AWeb PNG plugin View PNG images with support for transparency, progressive display, and scaling
AWeb JFIF plugin View JPEG/JFIF images with support for progressive display and scaling
AWeb GIF plugin View GIF images including animated GIFs, with support for transparency and scaling


Each plugin provides detailed documentation including installation instructions, configuration options, and parameter descriptions. Click on a plugin name above to view its documentation.
