CSS Support Summary

Fully Supported CSS1/CSS2 Properties

  • text-align - left, center, right, justify
  • font-family - comma-separated font lists with generic families (serif, sans-serif, monospace)
  • font-size - absolute sizes (xx-small to xx-large) and relative sizes (smaller, larger)
  • font-style - italic, normal
  • color - hex color values (#RRGGBB format)
  • text-decoration - none (for links to remove underlines)
  • padding - pixel values
  • background-color - hex color values (#RRGGBB format)
  • position - absolute, relative, static, fixed (parsed, rendering may be limited)
  • top, left - pixel and percentage values (parsed, rendering may be limited)
  • margin-right, margin-left, margin-top, margin-bottom - pixel and percentage values
  • width, height - pixel and percentage values (for table cells)
  • vertical-align - top, middle, bottom, baseline (for table cells)

Partially Supported

  • border - parsed but full rendering not implemented
  • border-radius - parsed but not applied (CSS3 feature)

Implementation Notes

  • All CSS properties are supported via inline styles (STYLE attribute)
  • CSS properties can be combined in a single STYLE attribute
  • Table cell styles (width, height, vertical-align, background-color) are applied to both the cell body and table cell structure
  • Link styles (text-decoration: none, color) are properly applied to anchor elements
  • Font properties (font-family, font-size, font-style) work on body elements and table cells
  • Color properties (color, background-color) support hex color codes in #RRGGBB format

Not Yet Supported

  • External stylesheets via <LINK rel="stylesheet">
  • CSS rules in <STYLE> tags (parsing exists but application is limited)
  • CSS selectors (class, ID, element selectors) - parsing exists but application is limited
  • CSS cascade and specificity
  • CSS pseudo-classes (:hover, :visited, etc.)
  • CSS inheritance
  • Additional CSS1/CSS2 properties not listed above
  • CSS3 features (except border-radius parsing)
