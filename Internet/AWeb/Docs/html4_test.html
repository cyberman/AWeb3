Frames (HTML 4.0)

Fully Supported

  • FRAMESET element with COLS and ROWS attributes - Fully supported (see html-ej.html#FRAMESET)
  • FRAME element with SRC, NAME, FRAMEBORDER, MARGINWIDTH, MARGINHEIGHT, SCROLLING, NORESIZE - Fully supported (see html-ej.html#FRAME)
  • NOFRAMES element for non-frame browsers - Fully supported (see html-ks.html#NOFRAMES)
  • Frame targeting with TARGET attribute - Fully supported (see html-ad.html#A and html-ad.html#BASE)

Test Examples

Frames require a separate frameset document. Here's an example structure:

<FRAMESET COLS="25%,75%">
  <FRAME SRC="nav.html" NAME="navigation" FRAMEBORDER="1" SCROLLING="yes">
  <FRAME SRC="content.html" NAME="main" FRAMEBORDER="1" SCROLLING="auto">
  <NOFRAMES>
    <BODY>Your browser does not support frames.</BODY>
  </NOFRAMES>
</FRAMESET>

Fully Supported - IFRAME

  • IFRAME (inline frames) - Fully supported in strict mode (see html-ej.html#IFRAME)
  • IFRAME attributes: SRC, NAME, WIDTH, HEIGHT, FRAMEBORDER, SCROLLING, MARGINWIDTH, MARGINHEIGHT, ALIGN, HSPACE, VSPACE - All supported

XHTML Support

Overview

AWeb now supports XHTML (eXtensible HyperText Markup Language), which is HTML reformulated as XML.

Fully Supported

  • XHTML DOCTYPE detection - Automatically detects XHTML documents and applies strict parsing mode
  • XML declaration support - Recognizes <?xml ...?> declarations
  • Self-closing tags - Supports <br />, <img />, <hr /> syntax
  • CDATA sections - Supports <![CDATA[...]]> for embedding unparsed content
  • XHTML Mobile Profile (XHTML-MP) - Automatically detected and supported

  • CDATA sections are properly extracted and displayed as text content
  • XHTML Mobile Profile (XHTML-MP) documents are automatically recognized

Compatibility Summary

HTML 4.0 Features Fully Supported

  • Document structure and DOCTYPE declarations
  • Text editing elements (INS, DEL)
  • Enhanced META and LINK elements
  • Frames and framesets (FRAMESET, FRAME, NOFRAMES)
  • IFRAME (inline frames)
  • Advanced table structure (THEAD, TBODY, TFOOT, COLGROUP, COL)
  • Form elements and basic attributes (FORM, INPUT, SELECT, OPTION, TEXTAREA, BUTTON)
  • OBJECT and PARAM elements
  • Scripting integration (SCRIPT, NOSCRIPT)
  • Event handlers on HTML elements
  • Character entities (HTML 4.0 set)

HTML 4.0 Features Partially Supported

  • STYLE element and basic CSS1 - Limited CSS rendering support
  • EMBED, BGSOUND - Tolerant mode only (browser extensions)

HTML 4.0 Features Not Supported

  • Full CSS1/CSS2 implementation
  • External stylesheets (limited support)
  • LABEL, FIELDSET, LEGEND elements
  • OPTGROUP element
  • INPUT TYPE="image"
  • Table accessibility attributes (SUMMARY, SCOPE, HEADERS, ABBR)
  • Form accessibility attributes
  • APPLET element (deprecated)
  • Internationalization features (limited)

XHTML Support (New)

  • XHTML 1.0, 1.1, and XHTML-MP - Fully supported
  • Self-closing tags - Fully supported
  • CDATA sections - Fully supported
  • XML declaration detection - Fully supported
  • XHTML DOCTYPE detection - Fully supported
