About Pages

AWeb supports special internal URLs using the "about:" protocol scheme. These pages provide browser information, utilities, and special pages similar to those found in other web browsers. You can use these URLs in your hotlist, bookmarks, or as links in web pages.

Note that these URLs are specific to AWeb and may not be recognized by other browsers.

Browser Information

about: or about:version
Displays information about AWeb, including version details, supported web standards, license information, and third-party component acknowledgements. This page provides comprehensive information about the browser and its capabilities.

about:blank
Displays a blank page. This is useful for creating empty pages or as a default target for links. The page contains minimal HTML structure but no visible content.

Utility Pages

about:home
AWeb's default home page featuring a search interface powered by BoingSearch and quick links to commonly used resources. This page can be set as your browser's home page in the preferences, or accessed directly via this URL. The page includes links to AWeb documentation and Amiga resources.

about:fonts
A comprehensive font test page that displays how different font families, sizes, and styles render in AWeb. This page is useful for:
  • Testing how fonts appear with your installed font set
  • Viewing font rendering for serif, sans-serif, and monospace families
  • Comparing different font sizes and styles
  • Checking special character rendering
The page includes sample text using classical Japanese haiku to demonstrate text flow and readability.

Usage Examples

You can use these URLs in various ways:
  • Set about:home as your browser's home page in the network options.
  • Add about: to your hotlist to quickly access browser information.
  • Use about:blank as a target for links that should open in a new window without loading content.
  • Bookmark about:fonts for quick access to the font test page.

Technical Details

The about: protocol is implemented as a plugin (about.aweblib). If you do not need the about: URLs you can remove the about.aweblib plugin.
