External Stylesheet CSS1/CSS2 Compatibility Test

This page tests AWeb's support for CSS1 and CSS2 properties via external stylesheets loaded via <LINK rel="stylesheet">. The styles are defined in css_test.css and applied using CSS classes. These tests mirror the inline style tests in css_test.html to verify that external stylesheets work correctly.
